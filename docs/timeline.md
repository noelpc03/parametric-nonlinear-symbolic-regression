## ✅ Completado

### Regresion simbolica con PySR
Pudimos hacer que la regresión simbólica funcione con PySR, logrando encontrar funciones que se ajustan a datos sintéticos generados con ruido. 

### Elección de función de pérdida adecuada
Se decidió utilizar una función de pérdida basada en una sigmoide para maximizar la cantidad de puntos dentro de una tolerancia ε, en lugar del clásico Error Cuadrático Medio (MSE). Esta elección se fundamenta en la necesidad de un objetivo suave que permita una mejora continua en la optimización, alineándose mejor con el objetivo de "matchear" puntos dentro de un umbral específico. Esta decisión se detalla en el documento [sigmoide_vs_mse.md](sigmoide_vs_mse.md).
